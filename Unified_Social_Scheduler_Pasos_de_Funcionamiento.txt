Unified Social Scheduler for Streamers
Pasos completos para el funcionamiento del sistema (MVP + Escalable)
1. Preparación del Proyecto
●
Definir alcance exacto del MVP (plataformas iniciales y tipo de contenido).
●
Decidir qué funcionalidades quedan fuera del MVP (auto-posting real, IA, pagos).
●
Crear repositorios separados para frontend, backend y documentación.
2. Backend – Base del Sistema
●
Inicializar proyecto Node.js con Express.
●
Configurar variables de entorno, CORS y middlewares.
●
Implementar autenticación JWT (registro, login).
●
Definir roles de usuario: USER y ADMIN.
3. Base de Datos
●
Configurar PostgreSQL (Neon).
●
Crear modelos: Users, Posts, Platforms, Licenses, Analytics.
●
Configurar Sequelize y migraciones.
4. Licencias y Control Administrativo
●
Panel admin para crear y asignar licencias.
●
Control de estado y fecha de expiración.
●
Validación de licencia en cada acción clave.
5. Frontend – Interfaz de Usuario
●
Crear aplicación React con Tailwind CSS.
●
Configurar Axios y rutas protegidas.
●
Crear dashboards separados para usuario y admin.
6. Programación de Contenido
●
Formulario de creación de publicaciones.
●
Selección de plataforma y fecha/hora.
●
Vista de calendario y filtros.

●
Estados de publicación (draft, scheduled, published).
7. Notificaciones y Onboarding
●
Notificaciones visuales con react-hot-toast.
●
Guía interactiva inicial con react-joyride.
8. Analítica Básica (MVP)
●
Conteo de publicaciones.
●
Publicaciones por plataforma.
●
Frecuencia de uso semanal.
9. Exportación de Datos
●
Exportar publicaciones en formato JSON.
●
Exportar estadísticas básicas.
10. Testing y Calidad
●
Pruebas de autenticación y roles.
●
Validación de licencias vencidas.
●
Pruebas de fechas inválidas y permisos.
11. Deploy y Producción
●
Backend desplegado en Railway.
●
Base de datos en Neon.
●
Frontend desplegado en Railway o Vercel.
●
Configuración de variables de entorno y CORS.
12. Escalabilidad Futura
●
Integraciones reales con APIs de redes sociales.
●
Auto-publicación.
●
Analytics avanzadas.
●
IA para sugerencia de horarios.
●
Aplicación móvil.
